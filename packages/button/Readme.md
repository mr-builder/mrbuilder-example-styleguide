





### Installation    
```sh
 $ yarn add button
```      


### Usage
In your javascript 
```js static
import Button from "button";
```
 

### Running
- start 
```sh
$ yarn run start
```

- test 
```sh
$ yarn run test
```
- karma 
```sh
$ yarn run karma
```
- build 
```sh
$ yarn run prepublish
```
