





### Installation    
```sh
 $ yarn add styleguide
```      


### Usage
In your javascript 
```js static
import Styleguide from "styleguide";
```
 

### Running
- start 
```sh
$ yarn run start
```

- test 
```sh
$ yarn run test
```
- karma 
```sh
$ yarn run karma
```
- build 
```sh
$ yarn run prepublish
```
